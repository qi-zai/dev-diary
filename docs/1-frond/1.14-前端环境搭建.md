### 安装后配置eslint

```json
{
    "editor.tabSize": 2,
    "eslint.validate": [
        "javascript",
        "html",
        "vue"
    ],
    "editor.codeActionsOnSave": {
        "source.fixAll.eslint": true
    },
}
```
